## 프로젝트 소개

날씨에 따라 어울리는 음악과 분위기가 달라진다는 점에서 출발한 프로젝트입니다.

사용자가 접속하면 현재 위치의 실제 날씨를 감지해서 그 날씨에 어울리는 배경과 문구, 추천곡을 한 화면에서 자연스럽게 보여줍니다. **맑음 / 흐림 / 비 / 눈** 네 가지 날씨마다 배경 사진, 날씨 아이콘, 안내 문구, 추천곡과 앨범 커버가 바뀌고 스크롤을 내리면 추천곡 상세와 가까운 다른 지역의 날씨까지 이어집니다.

전체적으로 글래스모피즘 스타일로 감성을 통일해 날씨 정보를 보는 것만으로도 기분 좋은 경험이 되도록 구성했습니다.

<br>

## 기술 스택

- **HTML5 / CSS3**
  - Flexbox 레이아웃 + CSS 변수 기반 스케일 시스템으로 반응형 구현
  - `backdrop-filter` 와 SVG 굴절 필터(`feDisplacementMap`)로 글래스모피즘 표현
  - 스크롤 위치에 비례한 배경 크로스페이드
- **JavaScript**
  - Geolocation + 실시간 날씨 연동
  - IntersectionObserver 기반 스크롤 등장 애니메이션
  - 오디오 플레이어 (재생 / 일시정지 / 무한 반복 / 진행바 / 자동재생)
- **외부 API**
  - `Open-Meteo` : 실시간 날씨 코드 및 기온
  - `BigDataCloud reverse-geocode` : 좌표를 한글 행정구역(시·도 / 시·군·구)으로 변환
  - `ipwho.is` : 위치 권한 거부 시 IP 기반 위치 폴백
- **폰트** : Pretendard (CDN) 사용

<br>

## 주요 기능 및 구현 내용

**1. 날씨 기반 첫 화면** : 접속 위치의 실시간 날씨(맑음 / 흐림 / 비 / 눈)에 따라 배경 사진, 날씨 아이콘, 타이틀 및 서브타이틀 문구가 자동으로 전환됩니다. 위치 칩에는 실제 지역명과 현재 기온이 표시됩니다.

**2. 추천 음악 & 뮤직 플레이어** : 날씨별 추천곡의 앨범 커버와 곡명을 보여주고, 상단 고정 플레이어에서 재생 / 일시정지, 무한 반복, 진행바 탐색, 곡명 슬라이드(마퀴), 자동재생(첫 상호작용 시 시작)을 지원합니다.

**3. 다른 지역 날씨** : 현재 위치에서 가장 가까운 다른 도시 3곳의 실시간 날씨를 카드로 보여줍니다.

**4. 시간대별 배경** : 접속 시각(아침 / 점심 / 저녁 / 밤)에 맞춰 마지막 섹션의 배경 사진이 바뀝니다.

**5. 스크롤 인터랙션** : 섹션이 넘어갈 때 배경이 크로스페이드로 부드럽게 이어지고, 요소들이 아래에서 위로 떠오르며 등장합니다.

**6. 반응형 레이아웃** : 세로 / 좁은 화면에서는 카드가 화면 중앙으로 정렬되고, 다른 지역 카드가 세로로 쌓이도록 레이아웃이 전환됩니다.

**7. 디버그 기능** : URL 끝에 `?clear` `?cloudy` `?rainy` `?snowy` 를 붙이면 해당 날씨 화면을 강제로 확인할 수 있습니다.
